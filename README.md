# Giffygram

Built on vanilla JavaScript, CSS, and HTML, Giffygram lets users post their favorite gifs. Each post can be favorited by other users, and deleted by the original author. Users can also send direct messages to each other, and be notified when they have messages.

### Starting the API

Open a terminal window and go to the `giffygram` directory.

1. `cd api`
1. `cp db.json.example giffygram.json`
1. `json-server giffygram.json`

### Starting the Web Server

Open another terminal window and go to the `giffygram` directory.

1. `cd src`
1. `serve`
